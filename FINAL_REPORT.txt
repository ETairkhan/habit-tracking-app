## 📞 Техподдержка

Если у вас есть вопросы:

1. **Приложение не запускается?**
   ```bash
   npm install                 # Переустановите зависимости
   npm run dev                 # Запустите оба сервера
   ```

2. **Ошибка при сборке?**
   ```bash
   cd client
   rm -rf dist node_modules    # Очистите кэш
   npm install
   npm run build
   ```

3. **Backend не отвечает?**
   - Проверьте что MongoDB запущена
   - Проверьте .env файл в server папке
   - Убедитесь port 5000 не занят

4. **Проверьте логи:**
   - F12 в браузере → Console tab
   - Network tab для API ошибок
   - Backend console для server ошибок
